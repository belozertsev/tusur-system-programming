Задание:
1. Подготовить Dockerfile, в котором:
	- осуществляется виртуализация операционной системы, соответствующей варианту задания (Debian)
	- устанавливаются необходимые для выполнения задания пакеты
	- запускается написанный bash-скрипт
2. Написать bash-скрипт, который должен:
	- при запуске выводить фамилию и имя автора, название программы и краткое описание
	- выводить подсказки для пользователя, помогающие пользователю корректно работать с программой
	- работать в режиме диалога с пользователем ("вопрос-ответ")
	- обрабатывать аварийные ситуации, ошибки и некорректно введённые данные, а сообщения об ошибках должны выводиться в поток stderr
	- быть организован в виде бесконечного цикла с возможностью выхода по запросу
	- завершаться с кодом возврата (1 или 0 в зависимости от того, выводилось ли на последней итерации цикла сообщение об ошибке)

Функционал bash-скрипта:
- запрашивает у пользователя путь к образу CD-диска
- имплантирует в образ CD-диска контрольную сумму его содержимого, если указанный образ CD-диска ее не содержит
- выводит значение контрольной суммы образа и проверяет ее корректность, если указанный образ CD-диска содержит контрольную сумму

Для выполнения задания использованы команды checkisomd5 и implantisomd5
